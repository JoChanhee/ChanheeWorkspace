# Theme Customization 🎨

The Soft Neumorphism design system is heavily reliant on CSS variables.

You can modify them in `styles.css`:

```css
:root {
  --color-primary: #ff6b6b; /* Playful Coral */
  --color-secondary: #4ecdc4; /* Soft Mint */
}
```

Try changing `--bg-primary` to a dark gray to see how the neumorphic shadows react! (Hint: You'll also need to invert `--shadow-light` and `--shadow-dark`).
