# Overview 📖

The **Markdown Reader** is a simple, lightweight, yet visually stunning tool for browsing markdown files. 

## Core Philosophy
We believe that reading documentation shouldn't be boring. By applying modern design principles like **Soft Neumorphism** and a **Playful Fun** color palette, we make learning and referencing code enjoyable.

### How it Works
1. A single `index.html` structure.
2. Lightweight vanilla JavaScript to parse `documents.json`.
3. Fetch API dynamically loads markdown content.
4. Rendered directly in your browser with zero build steps!
