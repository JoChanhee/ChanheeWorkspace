# Troubleshooting 🐛

If the Markdown Reader is acting strange, try these common fixes:

1. **Clear cache**: Browsers aggressive cache fetch requests sometimes.
2. **Local Server Setup**: Make sure you're using a proper http server and not raw `file://` protocol.
3. **Invalid JSON**: Check `documents.json` for trailing commas!

If the layout breaks, ensure your viewport width isn't triggering a weird breakpoint.
